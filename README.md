# 🎬 Movie Rating Analysis Dashboard (Netflix Style)

## 📌 Project Overview
This project is an interactive Netflix-style movie analytics dashboard built using Streamlit. It analyzes movie ratings and genres and provides simple movie recommendations.

---

## 🚀 Features
- Netflix-style UI dashboard
- Top rated movies visualization
- Genre-wise analysis
- Simple recommendation system
- Interactive filters (Streamlit sidebar)

---

## 📊 Dataset
- MovieLens Dataset (Kaggle)
- movies.csv
- ratings.csv

---

## 🛠️ Technologies Used
- Python
- Pandas
- Streamlit
- Matplotlib / Streamlit Charts

---

## 🤖 Recommendation System
A basic rating-based recommendation system suggests movies similar to user-selected ones.
